
Dark Video Sorter — это умный каталог и редактор ваших видеофайлов с расширенными возможностями для организации, редактирования и экспорта метаданных. Идеально подходит для любителей и профессионалов, желающих удобно управлять своей видеоколлекцией.
Возможности
Загрузка и управление видеофайлами

Простая перетаскиваемая зона для загрузки видео или выбор папки целиком
Поддержка любых типов видеофайлов
Автоматическая генерация превью для каждого видео
Возможность загрузки нескольких файлов одновременно

Автоматический парсинг метаданных

Извлечение года, актеров, стран и жанров из имени файла
Определение качества видео (например, WEB-DL, HDRip, BDRip) на основе имени файла и высоты видео
Определение формата аудио и наличия звука через анализ файла
Поиск изображений для обложки или установка собственной

Редактирование и настройка метаданных

Полное редактирование обложек, описаний, жанров, актеров, стран, годов и других характеристик
Множественный выбор стран, добавление и удаление
Ввод описания и сюжетов вручную
Обновление данных и сохранение изменений в localStorage

Фильтрация и сортировка

Быстрый поиск по названию или ключевым словам
Фильтрация по жанрам, актерам, странам
Сортировка по дате (новые/старые), названию (А-Я / Я-А)

Экспорт и импорт метаданных

Экспорт всей коллекции или отдельных метаданных в JSON-файл
Импорт метаданных для массового обновления или добавления информации
Возможность объединения данных с существующими

Удобный интерфейс

Интуитивно понятный дизайн на Tailwind CSS с тёмной темой
Анимации и hover-эффекты для приятного взаимодействия
Быстрый доступ к важным функциям через кнопки и контекстное меню

Дополнительные функции

Очистка всей базы данных (localStorage)
Изменение обложки каждого видео
Встроенный модальный плеер для просмотра видео
Поддержка drag-and-drop для загрузки файлов и папок

Технологии

HTML5, CSS3, Tailwind CSS
JavaScript (ES6+)
Использование localStorage для хранения данных
Поддержка загрузки файлов, генерации превью, обработка мультимедиа

Как использовать

Загрузите папку с видео или отдельные файлы через перетаскивание или клик по зоне загрузки.
Дождитесь автоматического парсинга и отображения метаданных.
Редактируйте информацию по необходимости — меняйте жанры, актеров, описания и обложки.
Используйте фильтры и сортировку для организации коллекции.
Экспортируйте метаданные или сохраняйте изменения автоматически.
Импортируйте метаданные для обновления или объединения коллекций.
